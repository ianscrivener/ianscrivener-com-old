Config Postgresql for local access only

2011 · 9 · 11

For Postgresql admin I use pgadminpgAdmin over SSH... never a bloated insucure option like phpPgAdmin.

When you install Postgresql its security is (rightly) locked down tight. So you need to open it up to actually be able to connect. Here's how you dot that;

1) Edit the pg_hba.conf file

nano /etc/postgresql/9.2/main/pg_hba.conf

2) Add a record to all IP connections form LOCALHOST only

host all all 127.0.0.1/24 md5

3) Restart postgresql

service postgresql restart
