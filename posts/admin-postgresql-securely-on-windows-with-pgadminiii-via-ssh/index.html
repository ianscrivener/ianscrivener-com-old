Admin Postgresql securely on Windows with pgAdminIII via SSH

2012 · 8 · 4

Intro


(1) Download & install pgadminIII

http://www.pgadmin.org/download/


(2) Download & install SecureCRT

http://www.vandyke.com/download/index.html


(3) Create a SSH tunnel for port 5432


(4) connect to posgresql with pgadmin

Note: if you need to access more than 1 posgresql database server then change the port numbers for the SSH tunnel & pgadminIII connection, eg 5433
